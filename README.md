# Ghana360

Ghana360 is a Flutter web application that provides a 360-degree view of Ghana, highlighting tourism, agriculture, mining, food, culture, and national heritage.

## Features
- Tourism destinations
- Agricultural information
- Mining sector insights
- Ghanaian food and culture
- Responsive Flutter Web interface
